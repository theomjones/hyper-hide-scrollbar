# hyper-hide-scrollbar

![travis status](https://api.travis-ci.org/theomjones/hyper-hide-scrollbar.svg?branch=master)

Hides the scrollbar in [Hyper](https://github.com/zeit/hyper).

## Install

Put `hyper-hide-scrollbar` in .hyper.js

```javascript
plugins: [
    // other plugins
    'hyper-hide-scrollbar'
],
```
